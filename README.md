# 🏏Virat Kohli Performance Analytics Dashboard


📌 Project Overview

This project analyzes Virat Kohli’s international cricket performance using Power BI.
The dashboard explores batting trends, opposition-wise performance, consistency metrics, and career progression using interactive visuals.

The objective of this project is to demonstrate sports analytics, data visualization, and storytelling with data using Power BI.

📊 Dashboard Pages

Executive Overview

• Career runs, matches, centuries and half-centuries

• Batting average and strike rate analysis

• Year-wise performance trends

Performance Analysis

• Format-wise performance comparison (Test, ODI, T20)

• Opposition-wise batting performance

• Batting position insights

Dismissal Analysis

• Distribution of dismissal types

• Pattern analysis of wickets


🛠 Tools & Technologies

Power BI,
DAX,
Power Query,
Data Modeling,
